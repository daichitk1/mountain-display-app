# 登山一覧表示アプリ
日本の主要な山を探せる図鑑アプリです。<br>
図鑑の中の山を選択すると、山の詳細を知ることができます。

こちらでデプロイしています。<br>
https://mountain-display-3tqrwrook-takei-daichis-projects.vercel.app/

# 利用したAPI
mountix API (1.1.0)　　https://mountix-docs.codemountains.org/#tag/mountains <br>
openAI API<br>
Google map API<br>
